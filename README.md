# Node.js + Vercel Form Project

## Install Dependencies

```bash
npm install
```

## Run Project

```bash
npm run dev
```

## MySQL Database

```sql
CREATE DATABASE nodeform;

USE nodeform;

CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(255),
    email VARCHAR(255)
);
```

## Deploy on Vercel

1. Push code to GitHub
2. Import repository into Vercel
3. Add environment variables
4. Deploy
